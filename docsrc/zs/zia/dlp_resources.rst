dlp_resources
==============

The following methods allow for interaction with the ZIA
DLP resources such as: `ICAP Servers`, `Incident Receivers`
`IDM Profiles`, and `EDM Schemas` endpoints.

Methods are accessible via ``zia.dlp_resources``
