site2site_vpn
=============

The following methods allow for interaction with the ZTB Site2Site VPN
(Cloud Gateway) API endpoints. Includes cloud gateway hubs, S2S VPN
connections, cluster gateways with interfaces, and S2S hub lists.

Methods are accessible via ``ztb.site2site_vpn``
