# Cyber Security Examples

Examples for querying cybersecurity incident data from Z-Insights.

## Available Methods

| Method | Description |
|--------|-------------|
| `get_incidents()` | Get cybersecurity incidents data |
| `get_incidents_by_category()` | Get incidents grouped by category |

## Usage

```bash
# Set environment variables
export ZSCALER_CLIENT_ID="your_client_id"
export ZSCALER_CLIENT_SECRET="your_client_secret"
export ZSCALER_VANITY_DOMAIN="your_vanity_domain"
export ZSCALER_CLOUD="beta"

# Run the example
python cyber_security_example.py
```
